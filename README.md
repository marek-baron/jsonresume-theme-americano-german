# Americano Theme

Americano is a German [JSON Resume](https://jsonresume.org/) theme based on the [Macchiato Theme](https://github.com/biosan/jsonresume-theme-macchiato).

## Why?

There are almost no German JSON Resume themes. I wanted a theme that is minimalistic, easy to read, printable, and tailored to German resumes.

## Usage

Install the theme:

```bash
npm install jsonresume-theme-americano-german
```

Render your resume with:

```bash
resumed render resume.json --theme jsonresume-theme-americano-german --output dist/resume.html
```
